# Windows Setup

Install **[Window Switcher](https://github.com/sigoden/window-switcher)** to allow switching between windows of the same application. Save the [window-switcher.ini](./window-switcher.ini) file in the same directory as `window-switcher.exe`. 

Place the [.wslconfig](./.wslconfig) file in the home directory.

Remap **Caps Lock** to **Left Ctrl** using the registry file.
